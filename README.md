# Start

## install project dependecies
```bash
yarn install
```

## Start project
```bash
yarn prisma generate

yarn start:dev 
```

## Docker up
```bash
docker-compose up -d
```

## Prisma Studio
```bash
yarn db:studio
```

## After starting project
 - Swagger - http://localhost:4200/swagger
 - PrismaStudio - http://localhost:5555 - opening automatically after run **yarn db:studio**
